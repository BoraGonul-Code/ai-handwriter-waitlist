# AI Handwriter – Waitlist

Landing page for the [AI Handwriter](https://ai-handwriter.onrender.com) early access waitlist.

## Stack
- Pure HTML + CSS + Vanilla JS (no framework, no build step)
- Deployed on Vercel

## Deploy
Connect this repo to Vercel. Select the root directory. No build command needed — Vercel serves `index.html` as a static site automatically.
